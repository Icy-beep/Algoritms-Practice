# Algorithms Practice 🚀

Репозиторий для практики написания алгоритмов на Python.

## 📋 Описание

<details>
<summary>Нажмите, чтобы развернуть описание</summary>
  
  ```bash
  `sum_even_numbers`:
  - Принимает список целых чисел.
  - Проверяет вхождение каждого числа в диапазон от -20,000 до 20,000.
  - Суммирует только четные числа.
  - Возвращает `1`, если итоговая сумма отрицательна (согласно бизнес-логике).
  - Строго валидирует типы данных (запрещает `bool`, `str` и другие типы внутри списка).
  ```

   ```bash
   `frequent_element`:
  - Принимает список целых чисел.
  - Проверяет вхождение каждого числа в диапозон от 1 до 20,000.
  - Собирает элементы в словарь по шаблону - `число`:`кол-во повторений`.
  - Проходит по словарю и определяет лидера по кол-ву повторений.
  - Возвращает 0 если список пустой.
  - Если несколько чисел встречаются одинаково часто, выбирает наименьшее из них.
  - Выбрасывает `TypeError`, если входные данные не являются списком или содержат не целые числа.
  - Выбрасывает `ValueError`, если число выходит за пределы диапазона [1, 20000].
  - Использует вспомогательную функцию `in_range` для валидации значений.
  ```
   
   ```bash
   `in_range`:
 - Проверяет, входит ли число в заданный диапазон (включительно).
 - Используется обеими основными функциями для валидации входных данных.
  ```

</details>

## 🛠 Установка
1. Клонируйте репозиторий:

   ```bash
   git clone https://github.com
   cd Algoritms-Practice
   ```

3. Создайте и активируйте виртуальное окружение:

   ```bash
   python -m venv .venv
   source .venv/bin/activate  # Для Linux/macOS
   .venv\Scripts\activate     # Для Windows
   ```

4. Установите зависимости для тестирования:

   ```bash
   pip install pytest pytest-cov
   ```

## 🧪 Тестирование

Для запуска тестов используйте команду:

   ```bash
   python -m pytest -v
   ```

Для проверки покрытия кода тестами:

   ```bash
   python -m pytest --cov=src tests/
   ```

## 🏗 Структура проекта

* `src/` — исходный код функций.
* `tests/` — модульные тесты на pytest с использованием параметризации.
* `main.py` — точка входа и тестирование производительности функции (написана с помощью ии).
