# Algorithms Practice 🚀

Репозиторий для практики написания алгоритмов на Python с фокусом на тестирование и профилирование производительности.

## 📋 Описание
На данный момент в проекте реализована функция `sum_even_numbers`, которая:
- Принимает список целых чисел.
- Проверяет вхождение каждого числа в диапазон от -20,000 до 20,000.
- Суммирует только четные числа.
- Возвращает `1`, если итоговая сумма отрицательна (согласно бизнес-логике).
- Строго валидирует типы данных (запрещает `bool`, `str` и другие типы внутри списка).
